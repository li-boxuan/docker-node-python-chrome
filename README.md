Docker image for Node.js automated tests. It also includes python3 and pip3.
A usecase is you have Django backend and Javascript frontend, and you
want to do end-to-end testing.

## Versions
- node: v10.12.0
- npm: 6.4.1
- python3: 3.4.2
- pip3: 1.5.6
- google-chrome: 70.0.3538.77
